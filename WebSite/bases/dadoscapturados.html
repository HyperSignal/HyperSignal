Quais dados nós capturamos no aplicativo?

Como nosso objetivo é manter a privacidade do usuário, apenas capturamos informações relevantes para estatística. Você poderá ver o arquivo que é gerado em texto puro os dados que serão enviados ao site. Um exemplo de arquivo pode ser visto abaixo:

D:-23.647473407909274 -46.65673613548279 16
T:-23.0 -46.0
DEVICE:GT-I9000B
MANUFACTURER:samsung
MODEL:GT-I9000B
BRAND:samsung
RELEASE:2.3.3
ANDROID:GINGERBREAD
SOFTVM:1.5a
SOFTVN:8
X:56865
O:VIVO
H:1d6b4531f08a889ea0fa974a2191a4d3

Onde:

  • D => Representa latitude, longitude e nível de sinal (0 a 31) do ponto correspondente.
  • T => Representa latitude e longitude da antena correspondente
  • DEVICE,MANUFACTURER,MODEL,BRAND,RELEASE,ANDROID => Modelo, fabricante, e versão do aparelho usado.
  • SOFTVM e SOFTVN =>Versão do Signal Tracker
  • X => Resultado de uma fórmula usada para validar o arquivo ( a fórmula tem como base apenas as latitudes, longitudes e sinais )
  • O => Operadora usada no aparelho
  • H => Hash para validação do arquivo ( Nela é usado todas as informações anteriores apenas )

Para maior privacidade, após o processamento todos os dados são armazenados com o nome de usuário SYSTEM, assim depois de processados, não haverá nenhum vestígio da informação que você enviou, apenas a contagem de quantos quilômetros de dados você enviou (visto no ranking ao lado da página). Apenas será armazenado o modelo do aparelho e o nome do usuário que enviou ele, assim como uma média de sinal por aparelho.

Lembre-se que todos os arquivos são enviados com a autenticação do Facebook. Porém apenas usamos o nome do usuário e Cidade/País para referência aos dados enviados. Nenhuma outra informação é usada.
